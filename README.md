## Overview

Dark-themed developer portfolio built entirely with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools. Clean, fast, and production-grade from the ground up.

## Features

| Feature | Details |
|---|---|
| Custom Cursor | Animated gold dot + ring, desktop only |
| Hero | Typewriter terminal animation, rotating headlines |
| About | Count-up stats, dynamic month calculator from Sept 2024 |
| Skills | Categorized tech stack with core tag highlights |
| Experience | Timeline with certificate modal viewer |
| Projects | Featured banner on main, full filtered list on projects page |
| Contact | Working Formspree form, social links, resume download |
| Responsive | Mobile hamburger nav with fullscreen overlay |
| Scroll Reveal | IntersectionObserver fade-up on all sections |
